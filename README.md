# Mi Portafolio Personal - Hoja de Vida

## 📋 Propósito del Proyecto

Este proyecto es un **portafolio personal / hoja de vida interactiva** desarrollado como parte del proyecto evaluativo de la asignatura **Ingeniería Web**. El objetivo principal es demostrar las habilidades adquiridas en desarrollo frontend utilizando tecnologías modernas como NextJS, React, TailwindCSS y TypeScript.

### Características principales:
- 📱 Diseño responsive que se adapta a diferentes tamaños de pantalla
- 🎨 Interfaz basada en diseño de Figma proporcionado por el profesor
- 💼 Sección de perfil con diálogo interactivo
- 🏅 Trayectoria deportiva y logros personales
- 🛠️ Habilidades técnicas y blandas
- 📚 Educación y conocimientos
- 🚀 Portafolio de proyectos con scroll horizontal
- 🔗 Enlaces a redes sociales (GitHub, LinkedIn)

## 🚀 Cómo Ejecutar el Proyecto

### Requisitos previos

Asegúrate de tener instalado:

- **Node.js** (versión 18 o superior)
- **npm** (viene con Node.js)
- **Git** (opcional, para clonar el repositorio)

### Pasos para ejecutar localmente

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/TU-USUARIO/mariana-martinez-portafolio.git
   cd mariana-martinez-portafolio

2. **Instalar dependencias**
   ```bash
   npm install

3. **Ejecutar en modo desarrollo**
   ```bash
   npm run dev

4. **Abrir en el navegador**
Abre tu navegador web (Chrome, Firefox, Edge, etc.) y ve a: http://localhost:3000
